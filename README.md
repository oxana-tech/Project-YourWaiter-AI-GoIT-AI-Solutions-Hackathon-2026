# Project-YourWaiter-AI-GoIT-AI-Solutions-Hackathon-2026

## Проєкт створений під час хакатону як масштабована AI-native SaaS платформа для HoReCa бізнесу.
## YourWaiter AI — це AI-помічник для закладів громадського харчування , який допомагає гостям:  
- отримувати персоналізовані рекомендації страв;
- спілкуватися з AI-офіціантом через QR-меню;
- отримувати рекомендації з урахуванням обмежень по здоров'ю та дієтичних вподобань;
- швидше та зручніше оформлювати замовлення.

Для закладів HoReCa платформа покращує:
- додаткові продажі;
- клієнтський досвід;
- операційну ефективність;
- швидкість обслуговування;
- зниження навантаження на персонал
- замість офіціанта бере на себе першу і найдовшу частину з огляду меню і визначення замовлення.

  Команда розробила:

- frontend-архітектуру;
- backend-інфраструктуру;
- AI-функціональність;
- Стратегію зростання і масштабування продукту;
- дизайн
- презентацію для інвестора;
- демо продукту.

# Бізнес-модель

YourWaiter AI використовує subscription-based SaaS модель для ресторанів із фокусом на:

- Додаткові продажі на базі штучного інтелекту;
- автоматизацію ресторанів;
- масштабований повторюваний дохід;
- Взаємодія з клієнтами на основі штучного інтелекту.
