# Project-YourWaiter-AI-GoIT-AI-Solutions-Hackathon-2026
YourWaiter AI — це AI-помічник для ресторанів, який допомагає гостям:  - отримувати персоналізовані рекомендації страв; - спілкуватися з AI-офіціантом через QR-меню; - отримувати рекомендації з урахуванням дієт та алергій; - швидше та зручніше оформлювати замовлення.
